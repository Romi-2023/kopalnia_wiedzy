# Kopalnia Wiedzy

Gamified educational web app built with **Python** and **Streamlit**.

**Kopalnia Wiedzy** turns learning into an interactive adventure.  
The app combines quizzes, missions, rewards and educational modules to help children learn through play.

It is designed as a larger multi-screen application with modular architecture, user progress, game-like mechanics and supervision features.

## Project Overview

Kopalnia Wiedzy is an educational app focused on engagement, progression and accessibility.

Instead of presenting knowledge as static content, the application guides the user through multiple learning areas, interactive tasks and reward systems. The project combines educational logic with product thinking and practical Python development.

The app has evolved beyond an early MVP and now uses a more modular structure with separate areas for routing, views, UI elements and app logic.
